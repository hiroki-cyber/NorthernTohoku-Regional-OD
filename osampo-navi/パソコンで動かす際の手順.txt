ご自身のパソコン（C:\Users\PC25054\Desktop\osampo-navi）で動かす際の手順
エクスポートしたファイルをパソコンのフォルダーに移したあと、ご自身のPCローカル環境でこのアプリを起動するには、以下のステップを実行してください。
Node.jsのインストール（未導入の場合のみ）
パソコンに Node.js（推奨: LTS版）がインストールされている必要があります。Node.js公式サイトからインストーラーをダウンロードしてインストールしてください。
コマンドプロンプト（ターミナル）の起動
Windowsのスタートメニューから「コマンドプロンプト（cmd）」を起動し、フォルダへ移動します：
code
Cmd
cd C:\Users\PC25054\Desktop\osampo-navi
部品（パッケージ）のインストール
以下のコマンドを実行して、必要な関連パッケージをダウンロードします：
code
Cmd
npm install
お仕事開始（開発用サーバーの起動）
以下のコマンドを実行すると、パソコン内でアプリが起動します：
code
Cmd
npm run dev
画面に表示される http://localhost:3000（または http://localhost:5173 等）にブラウザでアクセスすると、ご自身のPC上で全く同じアプリを動かすことができます！
何かローカル環境での起動や、スマホでの表示についてご不明な点がございましたら、いつでもお気軽にお声がけください！
