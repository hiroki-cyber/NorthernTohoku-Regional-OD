GitHubアカウントまたはVercelアカウントを作成
Vercelの公式サイト にアクセスし、無料アカウントを作成します（GitHubやGoogleアカウントでログインすると早いです）。
ファイルをアップロードして公開（CLIを使う場合）

code
Bash
cd C:\Users\PC25054\Desktop\osampo-navi
npm install -g vercel
